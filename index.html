The Process

It all starts with an idea. Students are given a concept to base a website on, and from
there they follow a workflow that allows them to quickly create visually stunning and
highly functional websites.

Step 1: Brainstorm
This is where it all begins. Bringing together ideas, sketching out initial wireframe plans, and thinking of how to include all the necessary content. Brainstorming is a crucial step, and the rest of the project is based on what you come up with at this stage.
Step 2: Wireframe
Usually you start off with a sketch of how you want the website to look. From there you bring this into Adobe XD and create a "low-fidelity" wireframe, without colours or pictures. As you gain more content, the colours and photos get filled in until you have exactly what you want your site to look like ready to reference.
Step 3: Code
Once your wireframe is ready, you create a GitHub file for your website, create all the needed folders and files, and then the coding begins. We use 3 main languages: HTML, CSS, and JavaScript. We start by creating the structure with HTML and a CSS grid, add function with JS, and finally we style it with CSS. After all this is done we run it through a validator to ensure a bug-free site.
Step 4: Content
This can be done before, after, or during coding, depending on the available time and the amount of people working on the project. This step is the creation and preparation of content for the website. Photography and videography, designing, and editing of this content are all done in this step.
